# Screenshots

此目录用于存放插件截图。

## 需要的截图

1. **banner.png** (1280x640) - 项目横幅/封面图
2. **popup.png** (800x600) - 配置界面截图
3. **translation.png** (1280x800) - 翻译界面截图
4. **settings.png** (800x600) - 设置界面截图

## 如何添加

1. 安装扩展
2. 使用截图工具截取
3. 保存到此目录
4. 更新 README.md 中的引用

## 截图工具推荐

- **macOS:** Shift+Cmd+5
- **Windows:** Win+Shift+S
- **Chrome:** Chrome 开发者工具 → 截图
