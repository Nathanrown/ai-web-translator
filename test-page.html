🌐 AI Web Translator Test Page

Welcome!

This is a test page for the AI Web Translator Chrome extension. You can use this page to test the translation functionality.

The extension supports multiple AI models including GPT-4, Claude, and Gemini. It translates content on-demand in a waterfall pattern.

Features

1. Side-by-side view - Original text on the left, translation on the right

2. Synchronized scrolling - Both panels scroll together

3. Hover highlight - Hover over text to highlight corresponding translation

4. Smart filtering - Code, formulas, and navigation are filtered out
